Step by step Procedure:

 

  1. Select the material of RTD you want to use.  Temperature coefficient (α) for the same will be displayed on the screen. Note this value.

 

  1. Click on ' R0' to get the value of R0 for selected RTD. Note the same.

 

  1. Click on 'Get Temp'. The temperature for which Rt is to be found will be displayed.

 

  1. Using  formula calculate the value of Rt and enter the answer in the box provided (upto 2 decimals with rounding off). Submit the answer using submit button.

 

  1. If your calculation is correct, go to step 3. Repeat the procedure min 3 times.

 

  1. If your calculation is wrong, you will be asked to repeat the same. Please refer to GET FORMULA tab and verify your calculations.

 

  1. After finishing minimum 3 set of correct readings, you can see the graph by clicking on 'Plot'.

 

  1. When the graph is displayed click Next tab to repeat the procedure with different reference resistance(R0) value and with  different materials.

 

  1. Minimum 3 calculations are necessary to plot the graph.

 

  1. Study the graphs for RTD performance  with different reference resistance values and different materials
