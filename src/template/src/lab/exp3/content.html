Acceleration of a Four-Bar Mechanism



Letα2,α 3,and α 4 be the angular accelerations of the links 2,3 and 4 respectively. Given
α 2 is the input angular acceleration of the link 2. Since the link lengths a, b and c are
known and for the given θ2 value,θ3 and θ4 can be determined using position analysis
and given ω2, the velocities ω3 and ω4 canbe found using velocity analysis as has
already been explained.
The angular accelerations can then be found by,
$$\alpha_2=\frac{d\varpi_2}{dt},\alpha_3=\frac{d\varpi_3}{dt},\alpha_4=\frac{d\varpi_4}{dt}$$ if the time data, t, is available. Otherwise, with the knowledge of α
2, there are alternate methods of solving for the accelerations of links 3 and 4 as given below
$$\alpha_3=-\frac{a\alpha_2\sin\left(\theta_4-\theta_2\right)-a\varpi_2^2\cos\left(\theta_4-\theta_2\right)-b\varpi_3^2\cos\left(\theta_4-\theta_3\right)-c\varpi_4^2}{b\sin\left(\theta_4-\theta_3\right)}$$ $$\alpha_4=-\frac{a\alpha_2\sin\left(\theta_3-\theta_2\right)-a\varpi_2^2\cos\left(\theta_3-\theta_2\right)-b\varpi_3^2-c\varpi_4^2\cos\left(\theta_4-\theta_3\right)}{c\sin\left(\theta_4-\theta_3\right)}$$
